This creates a problem, that the Jekyll's CSS rules override some Swagger CSS rules, which results in a broken Swagger UI. We can not solve it by encapsulating Swagger UI in an iframe, because it would break the navigation. Therefore, we solve it by disabling the Jekyll layout (see "layout: none" above) and manually injecting Just The Docs sidebar next to the Swagger UI.
