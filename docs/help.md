# Help

If you have any questions about the airfocus API or this documentation - you can ask the [developer community](https://community.lucid.co/peer-support-1?utm_source=developer-airfocus&utm_medium=lucid-community&utm_campaign=airfocus-dev-links&utm_content=help-ask-community) to get support!

If you have found a bug in the API or the schema, or this documentation - please report the issue in our [developer community](https://community.lucid.co/peer-support-1?utm_source=developer-airfocus&utm_medium=lucid-community&utm_campaign=airfocus-dev-links&utm_content=help-ask-community).

If you have general questions or need help with the airfocus product itself - please visit our [Help Center](https://help.airfocus.com) or [ask the community](https://community.lucid.co/peer-support-1?utm_source=developer-airfocus&utm_medium=lucid-community&utm_campaign=airfocus-dev-links&utm_content=help-ask-community) for help.
